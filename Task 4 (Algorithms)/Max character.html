Задача 3 (Max character):

Написать функцию findMaxCharacter(). Функция принимает строку и возвращает символ, встречающийся в этой строке большее количество раз (если таких символов больше одного, то они все отображаются в результате).

Пример:

findMaxCharacter(“Hi people”) // выведет в консоль “p”, “e”

findMaxCharacter(“Appearance”) // выведет в консоль “a”
