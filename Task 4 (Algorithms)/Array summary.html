Задача 5 (Array summary):

Написать функцию summarizeArray(). Функция принимает массив, состоящий из чисел, и возвращает объект, содержащий 4 параметра: min (минимальное число массива), max (максимальное число массива), sum (сумма всех элементов массив), avg (среднее арифметическое элементов массива).

Пример:

summarizeArray([1, 2, 3, 4, 5]) // выведет в консоль { min: 1, max: 5, sum: 15, avg: 3 }

summarizeArray([]) // выведет в консоль { min: null, max: null, sum: null, avg: null }

summarizeArray([10]) // выведет в консоль { min: 10, max: 10, sum: 10, avg: 10 }
