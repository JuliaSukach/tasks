Задача 9 (Sum of two):

Написать функцию sumOfTwo(). Функция принимает 2 параметра: array и sum. Необходимо вернуть массивы, состоящие из пар чисел, взятых из исходного массива, сумма которых (чисел) равняется параметру sum. Каждое из чисел исходного массива может быть использовано несколько раз.

Пример:

sumOfTwo([1, 2, 3], 3) // выведет в консоль [1, 2]

sumOfTwo([1, 2, 2, 3, 4], 4) // выведет в консоль [2, 2], [1, 3]
