Задача 8 (Reverse array):

Написать функцию reverseArray(). Функция принимает массив и возвращает новый, в котором элементы составлены в обратном порядке. Использовать встроенный метод Array.prototype.reverse() запрещается!

Пример:

reverseArray([1, 2, 3, 4, 5]) // выведет в консоль [5, 4, 3, 2, 1]

reverseArray([1, 2, 3]) // выведет в консоль [3, 2, 1]
