Задача 7 (Array chunking):

Написать функцию splitArrayOnChunks(). Функция принимает 2 параметра: array и size. Необходимо разделить итоговой массив array на части (массивы), в которых будет содержаться указанное в параметре size количество элементов. В случае, если size больше или равен длины массива array, то функция должна вернуть исходный массив.

Пример:

splitArrayOnChunks([1, 2, 3, 4, 5], 2) // выведет в консоль [1, 2] , [3, 4], [5]

splitArrayOnChunks([1, 2, 3, 4, 5], 3) // выведет в консоль [1, 2, 3] , [4 ,5]

splitArrayOnChunks([1, 2, 3, 4, 5], 6) // выведет в консоль [1, 2 ,3, 4, 5]
